About ProgramOS

ProgramOS is an advanced operating system simulation built using Python and Tkinter. It offers a graphical interface similar to traditional operating systems, featuring a start menu, taskbar, user authentication, and much more.

Features

  • Interactive Taskbar and Start Menu
  • Terminal, File Explorer, and Task Manager
  • Guessing Game to play
  • Login and User Management
  • Customizable User Interface
